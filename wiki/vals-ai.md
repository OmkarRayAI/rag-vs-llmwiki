# Vals AI

Independent AI model benchmarking platform. Tagline: "Independent Evaluation,
Unbiased Benchmarks." Many benchmarks built in-house.

## Benchmarks by domain

- **Coding** — SWE-bench Verified, Terminal-Bench 2.1, Vibe Code Bench,
  LiveCodeBench, IOI.
- **Finance** — Finance Agent v2, CorpFin v2, MortgageTax, TaxEval v2.
- **Healthcare** — MedCode, MedScribe.
- **Legal** — LegalBench.
- **Academic / math** — ProofBench, GPQA Diamond, MMLU Pro, MMMU.
- **Aggregate indices** — Vals Index, Vals Multimodal Index.

## Models tracked

Anthropic, OpenAI, Google, xAI, DeepSeek, Alibaba, Moonshot AI, MiniMax,
Cohere, zAI.

## Offerings

Leaderboards, per-model pages, side-by-side comparison, Model Guide, App
Reports, industry-filtered rankings, performance-over-time charts, Pareto
cost-vs-accuracy views, methodology documentation.

## Snapshot (2026-05-30)

Claude Opus 4.8 leads the Vals Index at 70.17%, followed by GPT 5.5 and
Claude Opus 4.7.
