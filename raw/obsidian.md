# Obsidian

Source: https://obsidian.md

Obsidian is a personal knowledge management application that operates on plain
markdown files in a local vault. It was created by Erica Xu and Shida Li and
released in 2020.

Obsidian uses the wiki-link syntax `[[Page Name]]` to connect notes. The graph
view visualizes the link network. Backlinks show inbound references.

Plugins extend functionality. Dataview lets users query frontmatter as a database.
Templater inserts dynamic templates. The Obsidian Web Clipper saves web pages as
markdown.

Files are stored locally as `.md` with optional YAML frontmatter. This
plain-text format makes vaults git-friendly and tool-agnostic. Obsidian is
commonly paired with LLM workflows because the underlying format is readable
and editable by any agent.
